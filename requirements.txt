# aidemo2026 — 共通環境の requirements
#
# qwen（テキスト生成）/ diffusers（画像生成）/ wordvec（単語埋め込み）を 1 つの環境で動かす。
# このファイル1個で Mac / Windows / GPU の有無、どれでもインストールできる。
#
# ============ インストール手順（この4ステップ）============
# 1) Python 3.12.10 で venv を作って有効化:
#      pyenv install 3.12.10                                          # 3.12.10 が無ければ（pyenv 使用時）
#      <3.12.10 の python> -m venv ~/.venvs/aidemo2026                # 例: ~/.pyenv/versions/3.12.10/bin/python
#      有効化  Mac/Linux : source ~/.venvs/aidemo2026/bin/activate
#              Windows   : ~\.venvs\aidemo2026\Scripts\Activate.ps1
#      python -m pip install -U pip
#
# 2) torch を先に入れる（OS/GPU で違うのはここだけ。★ torch は固定しない＝各環境に合う版が入る）:
#      Mac (Apple Silicon)        : pip install torch torchvision
#      Windows/Linux + NVIDIA GPU : pip install torch torchvision --index-url https://download.pytorch.org/whl/cu128
#      GPU 無し（CPU のみ）        : pip install torch torchvision
#      （exact な torch 版は固定しない。端末を複製する時は別途の「凍結ロック」で再現する）
#      ※ CUDA が 12.8 でない人は https://pytorch.org/get-started/locally/ で自分の版の行を使う
#
# 3) このファイルを入れる:
#      pip install -r requirements.txt
#
# 4) （任意）notebook の kernelspec は name=python3 なので、VSCode / Cursor は手順1の
#    venv を interpreter に選べばそのまま動く（追加登録は不要）。classic Jupyter で
#    一覧に出したい場合のみ: python -m ipykernel install --user --display-name aidemo2026
#
# ※ Colab はインストール不要（torch も含め最初から入っている）。手順は不要で、そのまま開いて実行する。
# ※ torch をこのファイルに書かないのは OS/GPU で入れ方が違うため（手順2で先に入れる）。
